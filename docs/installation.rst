Installation
============

This role requires at least Ansible ``v1.7.0``. To install it, clone it
to your `DebOps`_ project roles directory::

    git clone http://github.com/ganto/ansible-roundcube.git

.. _DebOps: http://debops.org/


Role dependencies
~~~~~~~~~~~~~~~~~

- ``debops.secret``

- ``debops.nginx``

- ``debops.php5``

- ``debops.mysql``
