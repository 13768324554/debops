Welcome to ginas!
=================

ginas is not a server. It is a set of Ansible_ playbooks and shell scripts
that allows you to configure and manage a variety of Debian_-based services
- all from a single server, or a set of servers in a datacenter.

.. _Ansible: http://ansible.com/
.. _Debian: http://debian.org/
