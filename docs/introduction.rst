Introduction
============

The ``debops.root_account`` Ansible role can be used to manage the machine's root account.

Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.0.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops.root_account
