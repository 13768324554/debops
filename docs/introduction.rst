Introduction
============

.. include:: includes/all.rst

The ``debops.apache`` role allows you to setup and mange the `Apache HTTP Server`_.


Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.0.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops.apache
