This [Ansible](http://ansible.com/) role allows you to install and manage
[Roundcube](http://roundcube.net/), a IMAP Web client written in PHP.

**This role is still in development. Use it at your own risk.**


### Role dependencies

* ``debops.secret``

* ``debops.nginx``

* ``debops.php5``

* ``debops.mysql``
